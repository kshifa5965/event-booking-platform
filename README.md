# RespiraHub – Event Booking Platform

## 📌 Project Overview

RespiraHub is a modern and responsive Event Booking Platform developed using HTML, CSS, and JavaScript. The platform enables users to browse upcoming events, search and filter events, view event details, and book tickets online through an interactive booking form.

The project is designed to provide a seamless and user-friendly experience across desktop, tablet, and mobile devices while demonstrating frontend web development concepts such as responsive design, DOM manipulation, dynamic content rendering, and form validation.

## 📖 How It Works

1. Users visit the homepage.
2. Browse available events.
3. Search events using the search box.
4. Filter events by category.
5. View event information.
6. Fill out the booking form.
7. Submit the form.
8. Receive booking confirmation message.

## 💡 Key Functionalities

### Search Events
Users can search events by typing event names.

### Category Filter
Users can filter events such as:

- Music
- Technology
- Business

### Ticket Booking
Users can register for events by providing:

- Full Name
- Email Address
- Phone Number
- Event Selection
- Number of Tickets

### Form Validation
The application validates required fields before booking confirmation.
